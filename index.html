О чем эта вики?

Linux - ядро операционной системы, составляющее основу операционных систем семейства Linux GNU/Linux, а также ряда операционных систем для мобильных устройств, в том числе Android.

Проект GNU - проект по разработке свободного программного обеспечения (СПО), является результатом сотрудничества множества отдельных проектов

Дистрибутив GNU/Linux — общее определение операционных систем, использующих ядро Linux, готовых для конечной установки на пользовательское оборудование.
